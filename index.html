My portfolio

Mentor Goal
Mentor Goal est un défi d'entreprise où on devait créer un blog pour aider les étudiants à faire leurs CV, lettre de motivation etc.On a utiliser strapi.io pour ajouter,modifier ou supprimer l'article qu'on publiait sur le site et du Html ainsi que du Css pour la structure du site web.
Hyrrulecastle
Hyrrule Castle est un mini-jeu rpg qui est créer totalement en typescript.Le concept était de créer un algorithme où l'héros devait battre 10 goblins et le boss final pour finir le jeu. On pouvait ajouter des capacités à notre personnages comme se soigner, des attaques spéciales(boule de feu). A chaque défaite de l'ennemie on monte d'étage mais on pouvait aussi mourir.
Picross
Picross est un jeu de réfexion où on devait chercher les bonnes cases à compléter pour finir le jeu en se référant aux chiffres sur les côtés. Mon projet était de refaire tout le jeu en langages C ce qui n'était pas si facile mais au final terminer. On devait créer un menu de jeu mais aussi un algorithme permettant d'avoir une grille de manière aléatoires mais aussi une option de jeu où la grille est déja prédéfinie. Lors du choix du mode de jeux une fenêtre se lance pour afficher la grille et lorsque le joueur fini de compléter les bonnes cases une animation de victoire s'affiche et le jeu se réinitialise.

Veille Technologique

Les frameworks cross-platform permettent aux développeurs de créer des applications pouvant fonctionner sur plusieurs plateformes (iOS, Android, Windows, etc.) à partir d'un seul code source. Parmi les plus populaires, on retrouve Flutter, React Native, et Xamarin. Ces frameworks réduisent le temps de développement et les coûts tout en offrant des performances proches des applications natives.

Ces outils sont particulièrement adaptés pour les startups et les projets nécessitant une compatibilité multi-plateformes rapide et efficace.
